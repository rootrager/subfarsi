## 🚀 راهنمای نصب و راه‌اندازی قدم‌به‌قدم (بسیار ساده)

برای اجرای برنامه، فقط کافیه این ۳ قدم رو به ترتیب انجام بدید تا هیچ اروری نگیرید:

### قدم اول: نصب پیش‌نیازهای پایتون
مطمئن بشید که پایتون (نسخه ۳.۸ به بالا) روی سیستمتون نصبه. ترمینال رو تو پوشه پروژه باز کنید و دستور زیر رو بزنید تا تمام کتابخونه‌های لازم (مثل CustomTkinter و OpenCV) نصب بشن:
```bash
pip install -r requirements.txt

قدم دوم: نصب موتور ترجمه آفلاین (Ollama) و مدل پیشنهادی

ما برای ترجمه، استفاده از Ollama رو پیشنهاد می‌دیم چون کاملاً رایگان، امن و آفلاینه.

۱. اول برنامه Ollama رو دانلود و نصب کنید.
۲. مدل پیشنهادی ما: ما به شدت مدل translategemma رو بهتون پیشنهاد می‌دیم. این مدلِ فوق‌العاده که بر پایه Gemma 3 ساخته شده (در نسخه‌های 4B، 12B و 27B در دسترسه)، یک قدم رو به جلو و بزرگ در دنیای ترجمه متن‌بازه. این مدل فوق‌العاده سبک، پرسرعت و برای ترجمه بین ۵۵ زبان بی‌رقیبه.
برای دانلود این مدل، ترمینال رو باز کنید و بزنید:
Bash

ollama run translategemma

(نکته بسیار مهم: قبل از اجرای برنامه SubFarsiPro، همیشه مطمئن بشید که نرم‌افزار Ollama تو پس‌زمینه سیستم شما در حال اجراست!)
قدم سوم: اجرای استودیو

حالا که همه‌چیز آماده‌ست، این دستور رو بزنید تا برنامه باز بشه:
Bash

python main_app.py

⚙️ راهنمای بخش تنظیمات (AI Configuration)

وقتی برنامه باز شد، تو پنل سمت راست (AI Configuration) می‌تونید همه‌چیز رو شخصی‌سازی کنید:

    Whisper Model (مدل رونویسی صدا): اینجا می‌تونید دقت و سرعت استخراج صدای انگلیسی رو تعیین کنید. مدل base سرعت فوق‌العاده‌ای داره و برای سیستم‌های معمولی عالیه. اگه سیستم قوی‌تری دارید می‌تونید روی small یا medium بذارید.

    Translation Provider (موتور ترجمه): اگه می‌خواید آفلاین کار کنید، این رو روی ollama بذارید. اگه کلید API دارید، می‌تونید سرویس‌های ابری رو انتخاب کنید.

    Model (نام دقیق مدل): در این کادر باید اسم مدلی که دانلود کردید رو بنویسید (مثلاً translategemma یا translategemma:latest).

    Target Language (زبان مقصد): روی همون Persian (Informal Tehrani) تنظیم شده تا خروجی محاوره‌ای و جذابی بهتون بده.

    Batch Size (حجم پردازش گروهی): این عدد نشون میده که تو هر مرحله، چند خط زیرنویس برای ترجمه ارسال بشه. بهترین و پایدارترین حالت بین ۴ تا ۸ هست. اگه سیستمتون خیلی قویه می‌تونید این عدد رو بالاتر ببرید.

SubFarsiPro v5.0 is an open-source, heavily optimized AI tool designed to transcribe and translate video subtitles into natural Persian (Informal Tehrani dialect). It features a modern 3-column GUI, background multithreading, OpenCV thumbnail extraction, and supports models like translategemma (Gemma 3) via Ollama for high-speed, zero-hallucination processing.
Install requirements: pip install -r requirements.txt, run Ollama, and start with python main_app.py.
